# スラッシュコマンドリファレンス

IPUT Esports Discord Bot で利用できるスラッシュコマンドの一覧です。

---

## `/verify`

学籍番号によるメンバー認証を行います。認証が完了すると、`未認証` ロールが外れ、`認証済み` ロールが付与されます。

| 項目 | 内容 |
|------|------|
| **説明** | 学籍番号で認証します |
| **必要な権限** | なし（誰でも使用可能） |
| **実装ファイル** | [`src/commands/verify.ts`](../src/commands/verify.ts) |

### オプション・サブコマンド

| 名前 | 種別 | 必須 | 説明 |
|------|------|------|------|
| （なし） | — | — | コマンド実行時にモーダルが表示されます |

### 実行フロー

1. `/verify` を実行すると、学籍番号入力モーダルが表示されます。
2. 6桁の学籍番号（例: `240001`）を入力します。
3. `tk{学籍番号}@tks.iput.ac.jp` 宛に認証コードが送信されます。
4. 「認証コードを入力」ボタンを押し、6桁のコードを入力します。
5. 認証が完了し、ロールが付与されます。

### 使用例

```
/verify
```

> 補足: 認証コードの有効期限は5分です。期限切れの場合は `/verify` からやり直してください。

## `/unverify`

指定したユーザーの認証を解除します。認証情報が削除され、`認証済み` ロールが外れて `未認証` ロールが付与されます。

| 項目 | 内容 |
|------|------|
| **説明** | 指定ユーザーの認証を解除します（管理者専用） |
| **必要な権限** | `Administrator` 権限、または `運営` ロールの所持 |
| **実装ファイル** | [`src/commands/unverify.ts`](../src/commands/unverify.ts) |

### オプション・サブコマンド

| 名前 | 種別 | 必須 | 説明 |
|------|------|------|------|
| `user` | User | Yes | 認証を解除するユーザー |

### 使用例

```
/unverify user:@username
```

## `/profile`

認証済みユーザーのプロフィール情報を表示します。

| 項目 | 内容 |
|------|------|
| **説明** | プロフィールを表示します |
| **必要な権限** | サブコマンドにより異なる（下記参照） |
| **実装ファイル** | [`src/commands/profile.ts`](../src/commands/profile.ts) |

### サブコマンド

#### `/profile activity`

大会参加履歴を表示します（現在は未実装）。

| 項目 | 内容 |
|------|------|
| **説明** | 大会参加履歴を表示します |
| **必要な権限** | なし |
| **オプション** | なし |

**使用例:**
```
/profile activity
```

#### `/profile user`

指定したユーザーのプロフィールを照会します。

| 項目 | 内容 |
|------|------|
| **説明** | 指定ユーザーのプロフィールを照会します（管理者専用） |
| **必要な権限** | `Administrator` 権限、または `運営` ロールの所持 |
| **オプション** | `user` (User, 必須): 照会するユーザー |

**使用例:**
```
/profile user:@username
```

### プロフィール表示項目

プロフィールには以下の情報が含まれます。

- ユーザー名（Discord 表示名）
- ユーザーID
- 学籍番号
- 入学年度
- 認証日時
- アカウント作成日
- ゲームロール（サーバーで付与されているゲーム関連ロール）

## 権限まとめ

| コマンド | 必要な権限 |
|----------|-----------|
| `/verify` | なし |
| `/unverify` | `Administrator` または `運営` ロール |
| `/profile activity` | なし |
| `/profile user` | `Administrator` または `運営` ロール |
